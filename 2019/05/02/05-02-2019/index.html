生活

今天中午吃了顿食堂卖的咖喱饭，肉非常少几乎没有味道感觉也没有我自己做的香。我感觉他们的问题主要是胡萝卜放的有点少然后土豆切的块大。尽管我看教程上的土豆都切得比较大但是人家炖的时间长，食堂的这个咖喱饭就明显感觉炖的时间不到位所以比起我做的其实还是差很多的。因为周四下午的这节课改成了在图书馆上所以决定以后每周四下午都去图书馆学习然后学习完回家做饭这样效率高一点。今晚打算炒蘑菇和牛肉。

学习

这放假后的第一周其实挺奇怪的，好多课都被各种原因花式取消。之前的python课老师来不了，后来python的tutorial因为火警取消了，图书馆有一天停电了不让进，然后今天的线性代数辅导课也因为场地占用问题取消了。周四开始就有INFO1111的tutorial了，今天下午上data课的时候我正巧问了问我一个朋友这个课考的难不难。结果他是裸考的说是很难。然后给我提到了一些题目我看都是当时上课的时候提过的内容，我全复习到了。这个课目前的情况就是因为涉及的内容太少太死板所以就像是政治一样完全是死记硬背，所以我今晚又刻意强化了一下记忆。

除此之外，我把算法栈和队列那一章的书看完了，下次就可以开始做题了。我感觉这章的概念的难度比之前几章要低，感觉我没有什么似懂非懂的部分，乐观估计做题应该效果还可以。
